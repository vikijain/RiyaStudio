# Riya Bhandari — Personal Website

Dual-identity portfolio for **Riya Bhandari** (Siemens Financial Services): professional story, interactive résumé, personal interests, and a vibe-coded arcade.

| Side | Pages | Theme |
|------|--------|--------|
| **Professional** | Career story, Resume | Teal |
| **Personal** | Life hub, Art, Travel, Music | Pink |
| **Vibe coding** | Arcade with HTML mini-games | Both |

## Open the site

```bash
# from RiyaStudio/Riya
npx --yes serve .
# or
python -m http.server 8080
```

**Start at:** `Riya/index.html`

## Structure

```
Riya/
  index.html          ← Dual portal
  professional.html   ← Career story + digital practice
  resume.html         ← Full résumé
  personal.html       ← Personal hub
  playground.html     ← Vibe-coded games
  art.html / travel.html / music.html
  styles.css, app.js, games.js, resume.css, resume.js
```

## Notes

- Professional = narrative & direction; Résumé = full facts for scanning/print.
- Site is presented as vibe-coded (AI-assisted), not outsourced.
- Resume source: `Resume-RB.docx` in repo root.
